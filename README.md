# Alarma

Para este tutorial se requiere:

1x Raspberry Pi Pico
1x Breadboard
1x LED
1x resistencia 330 Ohms
1x PIR (Sensor de movimiento pasivo) (+1 Opcional)
1x Buzzer
1 x JQ6500 (opcional)
1 x mini parlante (opcional)
cables
